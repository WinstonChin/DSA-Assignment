# DSA-Assignment

## Features

### Admin

1. Add a new board game
2. Remove a board game
3. Add a new member account
4. Display a summary of board games borrowed/returned

### Member
1. Borrow a game
2. Return a game
3. View Borrow / Return History
4. Read game reviews from others
5. Add a record

## Data Types

Linked List ADT
Array

## Algorithm

Merge Sort
Sequential Search
